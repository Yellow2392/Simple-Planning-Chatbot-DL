# Planning - Deep Learning

The following repository contains a basic implementation of a chatbox capable of solving logical planning tasks. 

## Quick Start

In order to test and run the model, install this library:
```bash
pip -q install --upgrade transformers accelerate bitsandbytes einops
```

To run the test module:
```bash
python dev_test.py
```

To create the submission for the task:
```bash
python submit.py
```

\* A [Google Colab](https://colab.research.google.com/drive/1lUmVylbBoJ1mgDW8O6llwidjhA2T9aMF?usp=sharing) notebook is provided in order to execute the previous steps in a Linux environment.
